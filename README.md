# Crypto Tracker App 

A real-time cryptocurrency tracking application built with modern Android development practices.

* **Language:** Kotlin
* **Pattern:** MVI (Model-View-Intent)
* **Dependency Injection:** Hilt
* **Networking:** Retrofit & OkHttp
* **Real-time:** Socket.io
* **Image Loading:** Coil
* **Asynchronous:** Coroutines & Flow
* **UI Components:** Jetpack Compose (Canvas for charts)
